Linux Installation:

1. git clone tensorflow/models repo
Inside tensorflow/models/research/
2. "apt install -y protobuf-compiler" in terminal
3. "protoc --version"
4. "protoc object_detection/protos/*.proto --python_out=."
5. "export PYTHONPATH=$PYTHONPATH:pwd:pwd/slim"
6. "python3 setup.py build"
7. "python3 setup.py install"
